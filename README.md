A lightweight & easy to use npm package for 2b2t. 

# How to install: 
`npm install 2b2t`

## Parameters:

**PrioQ:**
```
Function: PrioQ()
let prio = await PrioQ()
prio[1] - Players in PrioQ.
prio[2] - Estimated Time in PrioQ. 
```
